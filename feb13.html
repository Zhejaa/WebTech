Request Headers

Request headers are lines that sent by the client in HTTP protocol transaction.

LIST OF REQUEST HEADERS
Accept - this can be used to specify certain media types which are acceptable for the response. It can also be used to indicate that the request is specifically limited to a small set of desired types.

Accept-charset - content with different character sets. Used to indicate what character sets are acceptable for the response

Accept-language - file type extension. This is similar to Accept but restricts the set of natural languages.

Accept-encoding - priority based. Lists the content-encoding types which are acceptable in the response.

Authorization - this represents the authorization information.
