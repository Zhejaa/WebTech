Internet

  • Definition

    - a global system that connects computer networks around the globe using the Internet protocol

    - refers to the global information system that: (i) linked together by a globally unique addresses space based on the Internet Protocol(IP) (ii) supports communication using the TCP/IP suite or its subsequent extensions/follow-ons and/or other IP-compatible protocols & (iii) provides, uses of makes-accesible, publicly or privately.

  • History

    Vinton Gray Cerf - father of internet

  • WWW

    World Wide Web

    - created by Tim Berners-Lee in 1989

    - a collection of internet resources like images, video, audio, etc. that are accessible through browsers.
