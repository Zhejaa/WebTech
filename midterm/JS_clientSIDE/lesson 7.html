Mutator methods

  • - copyWithin - copies a sequence of array elements within the array
  • - fill - fills all the elements of an array from a start index to an end index with a static value
  • - push - add elements and increase length
  • - pop - remove last element and return value
  • - shift - remove first element and shift
  • - unshift - insert an element at the first position and shift
  • - sort - sort elements of the array
  • - splice - add/remove element from an array
  • - reverse - reverses the order of the elements of an array in place â€” the first becomes the last, and the last becomes the first

    Accessor methods

  • - concat - returns a new array comprised of this array joined with other array(s) and/or value(s)
  • - includes - determines whether an array contains a certain element, returning true or false as appropriate
  • - split - will produce an array
  • - join - joins all elements of an array into a string
  • - slice - extracts a section of an array and returns a new array
  • - lastIndexOf - returns the last (greatest) index of an element within the array equal to the specified value, or -1 if none is found
  • - toSource - returns an array literal representing the specified array; you can use this value to create a new array
  • - toString - returns a string representing the array and its elements
  • - toLocateString - returns a localized string representing the array and its elements
